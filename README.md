# Aplicação com Angularjs e Spring Boot 2

## Tecnologias

* Java 8;
* Arquitetura MVC;
* Design Pattern Facade;
* Design Pattern Repository;
* Angularjs 1.6.4;
* JSON Web Token;
* Spring Boot 2;
   *Spring Boot 2 Data;
   *Spring Boot 2 Security;
   *Spring Boot 2 Hibernate JPA Repository;
   *Spring Boot 2 Embedded Apache Tomcat;	
* Bootstrap 4;
* Maven;
* SL4j Logger;
* Banco de dados PostgreSQL 9;
* IDE Eclipse;
* IDE Visual Studio Code;
